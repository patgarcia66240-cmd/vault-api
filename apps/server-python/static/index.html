Fonctionnalités puissantes

Tout ce dont vous avez besoin pour gérer vos clés API en toute sécurité

Chiffrement sécurisé

Vos clés sont chiffrées avec AES-256 avant d'être stockées. Seuls vous pouvez y accéder.

Multi-providers

Support natif pour Custom keys, Supabase et IA providers. Unifiés dans une seule API.

Authentification JWT

Authentification robuste avec tokens JWT. Gestion complète des utilisateurs et sessions.

Base PostgreSQL

Base de données PostgreSQL fiable avec Supabase. Données structurées et query optimisé.

Analytics & Stats

Suivez l'utilisation de vos clés API avec des statistiques détaillées en temps réel.

Intégration Stripe

Plans FREE et PRO avec paiement sécurisé via Stripe. Upgrade en un clic.
